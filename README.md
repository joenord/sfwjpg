SFWJPG is utility to convert Seattle Filmworks proprietary format image files (SFW) to J-PEG (JPG). 

History: In the 1990s and early 2000s, one could get film developed via land mail at Seattle Filmworks, and for an extra charge,
they would include a floppy diskette or later, a CD-ROM containing digitial versions of the images developed.
Sold as digital images, they were delivered in a non-standard format and the photos could only be viewed with a proprietary 
SFW viewer program which was included on the floppy/CD-ROM and which executes only under DOS. 

SFWJPG converts the Seattle Filmwork files to J-PEG, recovering these images to viewable form.

Supported file types include
   .SFW Seattle Filmworks SFW93A, SFW94A.
   .PWP Seattle Filmworks SFW95B
   .PIC Overlake Photo Express
   .KQP Konica Quality Photo, .PWP) 

SFWJPG is a command line Windows utility (has no GUI), modeled on its UNIX herritage back to the late 1990s.

The SFW94A formatted images are the most common and these include inside the SFW source file the date that the 
film was developed. The SFWJPG converter adds EXIF tags for date of photo taken, based upon this information.
